# IMDB for Netflix
IMDB for Netflix is a chrome extension created by Cas van Dongen. This extension will add relevant IMDB data to the movie tiles on the Netflix website.

### Recources
 - [Typescript 2.1.5](https://github.com/Microsoft/TypeScript)
 - [ts-Loader](https://github.com/TypeStrong/ts-loader)
 - [Webpack](https://github.com/webpack/webpack)

### Dependencies
 - [Chrome 0.1.0](https://github.com/afc163/chrome)
 - [Bluebird 3.4.7](https://github.com/petkaantonov/bluebird)
 - [jQuery 3.1.1](https://github.com/jquery/jquery)
